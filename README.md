This project demonstrates the **four major Object-Oriented Programming (OOP)** concepts using a simple school management system written in **Python**.

The program allows users to register either a **Student** or a **Teacher**, showing how:
- **Abstraction** is implemented through the abstract `Person` class.
- **Inheritance** allows `Student` and `Teacher` to share and extend behaviors.
- **Encapsulation** hides sensitive data like student scores.
- **Polymorphism** lets both subclasses have unique versions of the same method `show_details()`.
